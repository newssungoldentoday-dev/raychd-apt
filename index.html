Install Ray Chad with one apt command.

raychd-apt is the official APT repository for Ray Chad. Add the repo, run apt install, and you're connected — no manual builds, no dependency chasing.

Get Started View on GitHub →
raychd-apt — install session
$ curl -fsSL https://raychd.ddns.net/apt/gpg.key | sudo apt-key add -
Adding GPG key... OK
$ echo "deb https://raychd.ddns.net/apt stable main" | sudo tee /etc/apt/sources.list.d/raychd.list
Repository added.
$ sudo apt update && sudo apt install raychd
Reading package lists... Done
Setting up raychd (1.0.0) ...
$ raychd --version
ray#chad v1.0.0 — ready.
